00. О документации

Разделы документации посвящены различным темам, с которыми вы столкнётесь при работе с IDE. Познакомимся с ними:

  • 00. Введение — пояснения к документации и краткий экскурс о работе с IDE;
  • 01. Работа с редактором — описание каждого шага работы с редактором IDE;
  • 02. Работа с игрой «Защита пасеки» — как работать со схемами из игры «Берлога: Защита пасеки» в IDE;
  • 03. Работа с Arduino — про взаимодействие Arduino и IDE;
  • 04. Дополнительные материалы — рекомендовано разработчиками для изучения таких тем, как инженерия, программирование, электроника.

Спасибо, что пользуетесь нашей IDE!

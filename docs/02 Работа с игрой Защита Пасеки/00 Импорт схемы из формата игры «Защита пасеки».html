02. Работа с игрой «‎Защита пасеки»

00. Импорт схемы из формата игры «Защита пасеки»

Для действия потребуется подключение к компилятору.

Откройте Главное меню на левой панели, выберите пункт Импорт…. В меню выбора файла нажмите на файл схемы, скопированной из игры «Берлога: Защита пасеки» и нажмите OK. При успешном чтении и преобразовании в основном поле появится схема, соответствующая записанной в файл.
